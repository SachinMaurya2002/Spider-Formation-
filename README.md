Here's the magic behind the stunning spider formation:

colors = ['red', 'purple', 'blue', 'green', 'orange', 'yellow']

t = turtle.Pen()

t.speed(40)

turtle.bgcolor('black')


for x in range(720):

    t.pencolor(colors[x % 6])
